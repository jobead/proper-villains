# Proper Villains

> *"villain" (n.) — from Latin villanus: villager. A word the powerful invented to demonize those who won't comply.*

A Proper Villain is a [Dangerous Professional](https://www.kalzumeus.com/2017/09/09/identity-theft-credit-reports/) who has decided which rules do not apply — and built a worldview around that.

They break rules that reinforce unjust hierarchy, that discriminate and oppress, that are immoral. They have a strict personal code they will never allow someone else to corrupt.

If your system calls principled defiance villainy — fine. You defined the class. We extended it.

## What This Is

This repo is a roster. Each file in `villains/` is a person — a unique instantiation of the `ProperVillain` class, declaring who they are, what they do, and which rules don't apply to them.

It's not an organization. There's no membership. No dues, no board, no bylaws. It's a shibboleth — a way to find your people.
